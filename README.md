# hack.party

Quarantine friendly party games where everyone is welcome and anyone can play.

## Project setup
```
npm install
```

### Run your unit tests
```
npm run test
```

### Lints files
```
npm run lint
```

### Lints and fixes files
```
npm run lint:fix
```
